# cursor_clone
This is a frontend UI clone created for learning and practice purposes only.

**Tech Stack**

HTML5

CSS3 (Flexbox + modern layout techniques)

No frameworks

No JavaScript

No external UI libraries

**Features**

Desktop only landing page(no responsiveness yet).

Center-aligned sections like the original Cursor site

Hero section with CTA

Changelog section


Recent highlights / blog section

Final CTA section

Footer with multi-column links
